## Archive Policy

**When to Archive:**
- One-time setup guides (after setup is complete)
- Completion summaries (after feature is done)
- Templates that aren't actively used
- Documentation that has been consolidated into other docs

**When NOT to Archive:**
- Active feature documentation
- Reference documentation
- Setup guides that are still used
- Planning documents for future work
